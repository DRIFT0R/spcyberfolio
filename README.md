# Stephano Pereira Portfolio

A minimalist portfolio website for my IT and cybersecurity projects, technical writing, and professional background.

The site is built with vanilla HTML, CSS, and JavaScript and is deployed with GitHub Pages.

## Sections

- **Home** — featured cybersecurity and IT projects
- **About** — background, technical interests, tools, and certifications
- **Projects** — project archive with search
- **Writing** — selected technical writing and publications
- **Contact** — professional contact links

## Featured Work

The portfolio currently highlights projects involving:

- enterprise infrastructure and network security
- risk assessment and security documentation
- security tooling
- technical writing on privacy, surveillance, and cybersecurity
